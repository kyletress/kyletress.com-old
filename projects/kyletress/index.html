kyletress.com

Kyletress.com is my personal website which has been online in one form or another since 2002. I’ve used just about every popular CMS over the years, from simple static HTML files to ExpressionEngine, Harmony, WordPress, Statamic, and finally back to static files (though powered by Siteleaf this time).

Siteleaf published flat HTML. Posts are written in markdown, and I use GitHub to control the source code. I publish the site to GitHub pages, and I use CloudFront as my CDN.

Content and design are the two biggest challenges I’ve faced over the years with my personal site. I knew I never wanted the site to be a blog. I was looking for something more substantial. These challenges led me to draft up a set of brief rules guiding my personal website decisions.
