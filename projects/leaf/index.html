Leaf

web development, web design

I have been using Siteleaf to power my personal website for several months. It’s an amazing service, but I have one complaint – they don’t have an iOS app. They do have a terrific API, so I decided to learn iOS development by building an app to interact with it. Like many of my projects, this is an example of dogfooding; building something I need and want to use myself.

The app makes use of all the available API endpoints. Basic Authentication is needed to validate each request, and I store the user’s credentials in the Keychain.
