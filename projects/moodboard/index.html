Moodboard

Moodboard is an iPad application I co-created in 2010. I was responsible for the user interface design pre-iOS 7. It is available on the App Store

History

I wanted to create a digital mood board tool long before the iPad was released, but I considered the screen of the iPhone to be too small. When Apple released the iPad in 2010, it was a perfect match. The app was available on the App Store launch day.

Features

  • Custom color palettes
  • Very high resolution images
  • Customizable background images
  • Sharing
  • PDF and PNG export
  • Web and Photo Library import

Future

I haven’t worked on Moodboard since the release of iOS 7. I hope to see the interface redesigned to reflect the current Human Interface Guidelines.
